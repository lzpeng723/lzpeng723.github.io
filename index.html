Minimal

极简后台管理系统

快速上手 →

前后端分离

提供前端 Vue、后端 Spring Boot 完全分离的后台管理系统基础模板。

微服务版

后端采用 Spring Cloud Alibaba 第二套微服务组件(Nacos, Sentinel...)进行构建。

Vue 全家桶

前端使用 Vue, vue-element-admin, ElementUI, axios。

代码生成器

只需写好实体类即可在线生成前后端代码，增删改查导入导出等直接使用。

基础功能完善

内置用户、角色、菜单、部门、岗位、通知、监控、代码生成等基础功能。

持续更新

Minimal Cloud 还在持续更新, 计划实现工作流、报表模块。
